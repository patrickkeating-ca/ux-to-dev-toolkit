# Reference Documentation

This folder contains reference materials and templates for the UX-to-Dev Toolkit curriculum.

## Contents

- **pr-template.md** - Pull Request description template for professional GitHub collaboration

## Usage

These are reference materials students can use throughout the course and in their professional work.

### When to Use the PR Template

The PR template provides a structured format for creating clear, professional pull request descriptions. Students should reference this when:
- Creating their first pull request (Week 5)
- Working on collaborative projects
- Submitting work for review
- Building professional development habits

### For Instructors

Place additional reference materials, templates, and supplementary content in this folder to keep the main lesson folders focused on core learning content.
