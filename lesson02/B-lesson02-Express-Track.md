## Part 2: Learn These 5 Shortcuts (3 minutes)

In VS Code, practice each one once:

**1. Command + P** - Quick open file
- Opens file search
- Type filename, press Enter
- Fastest way to navigate

**2. Command + S** - Save file

**3. Control + `** - Toggle terminal
- Opens/closes terminal inside VS Code
- Backtick key (left of 1)

**4. Command + Shift + V** - Preview markdown
- Shows formatted view
- Essential for markdown work

**5. Command + B** - Toggle sidebar
- Hides/shows file explorer
- Gives you more screen space

**Practice these 5 until they feel natural.**

---

## Part 3: Change One Setting (2 minutes)

**Enable Auto Save:**
1. Press Command + , (Command + comma)
2. Search: "auto save"
3. Change to: "afterDelay"
4. Close settings

**Why:** Files save automatically.

---

## Quick Practice (3 minutes)

**Test your setup:**
```bash
cd ~/Desktop
mkdir vscode-test
cd vscode-test
code .
```

**In VS Code:**
1. Create new file: test.md
2. Type some markdown
3. Press Command + Shift + V to preview
4. Press Command + S to save
5. Press Control + ` to open terminal
6. Type `ls` in terminal - see your file

**If all that works, you're ready.**
